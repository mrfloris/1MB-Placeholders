# 1MB Placeholders

Little plugin for 1MoreBlock.com, which helps us to quickly add some global placeholders that we can use in various plugins to save us time during updates.

## Requirements

- PlaceholderAPI version 2.11.4-DEV-174 or newer (https://ci.extendedclip.com/job/PlaceholderAPI/)
- PaperMC version 1.19.4 or newer (https://papermc.io/downloads/paper)

## Installation

- Dump .jar in /plugins/ and start the server, let it load, and /stop the server again.
- Edit the config.yml of this newly created file, optionally (re-)add addtional placeholders.
- Customize other plugins and locations where these new global placeholders can be used.
- Start the server again.
